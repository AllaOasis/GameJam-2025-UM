## 3. Concept Overview
*Chrismes in Wintertown* is an exploration-based game where the goal is to find all 12 hidden Christmas gifts.  
The player explores a snowy world made up of three islands: a lighthouse and dock area, a mountain island, and a main village island with a decorated tree and a large snowman.  
The game uses a low-poly, medieval-inspired art style.  
The theme **Winter Is Coming** is interpreted through environmental hazards and a town prepared for winter and Christmas.

---

## 4. Key Mechanics
- World exploration and gift collection  
- Door interaction  
- Environmental hazard system (cold, heat, water, fire)  
- UI displaying health, cold, and heat levels
